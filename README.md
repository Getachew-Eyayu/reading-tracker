📚 Book Collection Manager (CLI Project)

A simple Python-based command-line book management system that helps users organize their reading habits by tracking books they want to read and books they have already read.

🚀 Features

📖 Add new books to your collection

✏️ Edit book titles

❌ Remove books from the system

✅ Mark books as read

🔍 Search for a book in your collection

📊 View reading summary statistics

⚠️ Low stock alert for unread books

📘 Display all books in organized format


🛠️ Technologies Used
Python 3
Data Structures (Sets, Dictionaries)
Functions and Modular Programming

The purpose of this project is to practice:
